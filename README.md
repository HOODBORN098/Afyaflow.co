# AfyaFlow — Modern Hospital Management System

AfyaFlow is a premium, feature-rich hospital management system designed to streamline clinical workflows, patient management, and administrative operations.

## 🚀 Core Features

### 🏥 Clinical Workflow
- **Real-time Queue Management**: Dynamic patient prioritization and department-specific queues.
- **Advanced Referrals**: Seamless transfer of care between departments and specific specialists with persistent data synchronization.
- **EMR & Vitals**: Comprehensive electronic medical records with real-time vitals tracking and prescription history.

### 🛡️ Security & Identity
- **JWT Authentication**: Secure, token-based access with role-based permissions (Admin, Doctor, Receptionist).
- **Persistent Profiles**: Real-time profile updates (email, username, department) that sync across all sessions.
- **Security Visibility**: Enhanced password fields with visibility toggles for improved user experience.

### 📊 Operations & Analytics
- **Inventory Management**: Track medical supplies and equipment with real-time stock alerts.
- **Ward Management**: Interactive hospital bed occupancy map with admission/discharge workflows.
- **Audit Logging**: Full traceability of all critical system actions for compliance and accountability.

## 🛠️ Technology Stack

- **Backend**: Spring Boot 3, Java 17, Spring Security (JWT), MySQL.
- **Frontend**: React 18, TypeScript, Vite, Tailwind CSS.
- **Design**: Material Symbols, Glassmorphism UI, Responsive Layouts.

## 📦 Getting Started

Please refer to the [**SETUP.md**](./SETUP.md) for detailed instructions on how to run the project locally.

1. **Clone the repository**
2. **Setup MySQL database**
3. **Run Backend** (`./mvnw spring-boot:run`)
4. **Run Frontend** (`npm install` && `npm run dev`)
